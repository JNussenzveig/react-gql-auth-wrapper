# react-auth-wrapper

> React auth wrapper &amp; components

## Usage

```jsx
import React, { Component } from 'react'

import { AuthScreen } from './src/AuthScreen'

class MyApp extends Component {
  render () {
    return (
      <AuthScreen />
    )
  }
}
```
